Preparation¶

Define the problem¶

Knowing from a training set of samples listing passengers who survived or did not survive the Titanic disaster, can our model determine based on a given test dataset not containing the survival information, if these passengers in the test dataset survived or not.

Other hints¶
  1. Total survival rate: 32%.
  2. Some groups of people were more likely to survive that others, such as women, children, and the upper-class.
Notes on code block above¶

  1. seaborn: a Python visualization library based on matplotlib.

  2. df = pd.read_csv(<path>): reads .csv file into the DataFrame df.

  3. To access data on Kaggle, use path "../input/<file>".

  • The general approach here to plotting here is called "small multiples", where the same kind of plot is repeated multiple times, and the specific use of small multiples to display the same relationship conditioned on one or more other variables is often called a trellis plot.

    1. Initialize the FacetGrid object with the dataset and the variables that are use to structure the grid.
    2. Then one or more plotting functions can be applied to each subset by calling FacetGrid.map() or FacetGrid.map_dataframe().
    3. Finally, the plot can be tweaked with other methods to do things like change the axis labels, use different ticks, or add a legend.
  • Parameters

    • row, col: strings; subsets of data
    • size, aspect: scalars, size represents height in inches; aspect is the aspect ratio, i.e. aspect * size gives the width of each facet in inches.

Deal with missing values¶
Three strategies:¶
  1. Generate a random number in the range $\mu \pm \sigma$.
  2. Guessing missing values with help of other correlated features.
  3. Combine method 1 and 2: instead of guessing based on median, we use random numbers in the range $\mu \pm \sigma$, based on combinations of correlated features.

for dataset in combine:
    for i in range(0, 2):
        for j in range(0, 3):
            
            # loop over `Sex` and `Pclass` to remove all empty values in column Ages
            # guess_df is a series indexed by arrange(n) 
            guess_df = dataset[(dataset['Sex'] == i) & \
                                  (dataset['Pclass'] == j+1)]['Age'].dropna()
            
            # compute median of guess_df 
            age_guess = guess_df.median()
            
            # we fill the corresponding guess_age
            guess_ages[i,j] = int( age_guess/0.5 + 0.5 ) * 0.5
    
    # print(guess_ages)
    
    for i in range(0, 2):
        for j in range(0, 3):
            
            # we replace missing values in each group by guess_ages[i,j]
            dataset.loc[ (dataset.Age.isnull()) & (dataset.Sex == i) & (dataset.Pclass == j+1),\
                    'Age'] = guess_ages[i,j]

    # convert Age to integer
    dataset['Age'] = dataset['Age'].astype(int)
